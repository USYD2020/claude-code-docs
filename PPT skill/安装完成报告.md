### 基本使用命令

```bash
./run.sh --plan <你的计划文件.json> --style styles/gradient-glass.md --resolution 2K
```

**参数说明**:
- `--plan`: PPT 内容规划 JSON 文件（必需）
- `--style`: 样式模板文件（可选，默认使用项目内置样式）
- `--resolution`: 输出分辨率，可选 2K 或 4K（可选，默认 2K）
- `--output`: 输出目录（可选，默认为 outputs/时间戳）

## 💡 API 配额问题解决方案

### 方案 1: 等待配额重置
- **免费层级配额**通常每天重置
- 建议明天再试，或等待几小时后重试
- 可以访问 [Google AI Studio](https://ai.dev/rate-limit) 查看当前配额使用情况

### 方案 2: 创建新的 API 密钥
1. 访问 [Google AI Studio](https://aistudio.google.com/apikey)
2. 使用不同的 Google 账号创建新的 API 密钥
3. 更新环境变量：
```bash
export GEMINI_API_KEY="你的新密钥"
```

### 方案 3: 升级到付费计划
- 访问 [Google Cloud Console](https://console.cloud.google.com/)
- 启用计费账户以获得更高的配额限制

## ⚠️ 重要注意事项

### 1. API 密钥安全
- ✅ 已将密钥添加到 `~/.zshrc`，每次打开新终端会自动加载
- ⚠️ 不要将 API 密钥提交到 Git 仓库
- ⚠️ 不要在公共场合分享你的 API 密钥

### 2. 虚拟环境使用
- 每次使用工具前必须激活虚拟环境
- 激活命令：`source venv/bin/activate`
- 退出命令：`deactivate`

### 3. 文件路径问题
- 由于目录名包含中文和空格，命令行中需要使用引号
- 建议使用绝对路径避免路径错误

### 4. 配额限制
- 免费层级有每日和每分钟的请求限制
- 建议合理安排使用，避免频繁调用
- 可以监控使用情况：https://ai.dev/rate-limit
