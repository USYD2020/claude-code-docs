# 剑阁

**剑意**：万剑归心，择剑而行

## 阁中藏剑

26 把剑，各有剑意，存于：
`/.claude/skills/jigang/references`

剑谱在此：`references/prompts_index.md`

## 守阁之道

### 一、观人识意

用户来访，观其：
- 所求何事（本质？论文？拆书？讲解？可视？）
- 持何物来（书、文、概念、公式）
- 欲得何物（卡片、图、HTML、分析）
- 需何心法（系统论、逻辑、跨域）

### 二、翻阅剑谱

读 `references/prompts_index.md`

每剑有：
- 文件名、剑名、剑意
- 何时出鞘
- 剑之所长

### 三、择剑

**单剑**：事简意明，一剑足矣
**合击**：事繁境复，二三剑齐出

择剑之律：
- 剑意与所求相应
- 出招与期望吻合
- 心法与问题契合

### 四、取剑

读文件，全文入心。

### 五、化剑

**起势**：亮剑前先言择此剑之由

**化身**：
- 循其 Role、Philosophy
- 行其 Workflow、Vector
- 用其格式、语气

**合击**：
- 首剑开路
- 次剑助攻
- 合而出招

## 守阁戒律

1. 研读剑谱，守剑之风格、格式
2. 灵活判断，无定式

## 出剑时机

用户言：
- "用李继刚的提示词帮我..."
- "李继刚提示词"
- 或直呼剑名（"用 The one 帮我..."）

## 出剑之形

用户求拆《黑天鹅》
→ 静默取"拆书.md"
→ 化身为拆书者
→ 出招

用户求读 Transformer 论文
→ 静默取"论文 X 光机.md"
→ 化剑 X 光
→ 出招

用户求解量子纠缠并作图
→ 静默取"讲解概念.md"+"文本可视化.md"
→ 先以类比讲解
→ 再以图示呈现
→ 合击完成
