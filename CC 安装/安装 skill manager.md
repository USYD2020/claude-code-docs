github-to-skills - 将 GitHub 仓库自动转换为 AI skills
skill-manager - 管理 skills 的生命周期（检查更新、列表、删除）
skill-evolution-manager - 根据用户反馈演进和优化 skills

安装完成后，你将能够使用以下命令：

/github-to-skills <github_url> - 从 GitHub 仓库创建新 skill
/skill-manager check - 检查 skills 更新
/skill-manager list - 列出所有已安装的 skills
/skill-manager delete <name> - 删除指定 skill
/evolve - 演进和优化 skills
这 3 个 skills 形成了一个完整的 skill 生命周期管理系统。
